# KonterQ Backend / CMS

This is the fullstack apps of konterQ

## Setup / Install

First you need install adonis-cli.

```bash
npm i -g @adonisjs/cli
```

after that clone this project, and run.

```bash
yarn initProject
```

next step is run the adonis

```bash
adonis serve --dev
```
